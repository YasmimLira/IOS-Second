TCC

Como parte do meu Trabalho de Conclusão de Curso (TCC), desenvolvi um projeto inovador que visa conectar pessoas em busca de serviços com prestadores de serviços locais. O desafio era criar uma plataforma que fosse amigável para o usuário e eficaz para ajudar as pessoas a encontrarem os serviços que precisam, ao mesmo tempo em que oferecem uma oportunidade de renda para prestadores de serviços locais.

Solução

Para abordar esse desafio, utilizei uma variedade de tecnologias, incluindo HTML, CSS e JavaScript, e optei por implementar o aplicativo como uma aplicação web responsiva baseada no framework React. Essa escolha de tecnologia permitiu criar uma interface de usuário intuitiva e de fácil navegação.

A plataforma oferece recursos abrangentes, incluindo perfis de prestadores de serviços, categorização de serviços, sistema de avaliações e agendamento de compromissos. Além disso, desenvolvi um sistema de gerenciamento de tarefas eficiente para que os usuários possam acompanhar e gerenciar suas solicitações de serviço de forma transparente.
